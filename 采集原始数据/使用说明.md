## 1. 单文件分析

```bash
python src/acquisition_analysis.py "F:\1\依怕热古丽\采集原始数据\AGC控制指令 361.csv"
```

此命令用于快速检查单个过程采集文件的结构和统计信息。

## 2. 目录批量分析

```bash
python src/acquisition_analysis.py "F:\1\依怕热古丽\采集原始数据"
```

此命令会遍历目录中的所有 CSV 文件，并输出每个文件的分析摘要和类别识别结果。

## 3. 生成完整报告

```bash
python src/acquisition_analysis.py "F:\1\依怕热古丽\采集原始数据" --report-dir output
```

执行后会生成：

- `acquisition_summary.json`
- `metric_summary.csv`
- `top_metrics.png`
- `lag_correlation.csv`
- `lag_correlation.png`

## 4. 输出文件用途

- `JSON`：适合保存结构化摘要
- `CSV`：适合保存类别统计和相关分析表
- `PNG`：适合放入文档或汇报材料

## 6. 常见问题

### 6.1 文件名分类不准确

当前分类主要依据文件名关键词，因此“其他类”数量较多是正常的。若要提高准确性，需要结合原始业务表或人工标注。

### 6.2 滞后相关结果为空

这通常表示两个变量在时间上没有足够重叠，或者采样粒度差异较大。可以尝试换另一组变量，或者先做时间重采样。

### 6.3 目录中有中文或特殊字符文件名

脚本已经尽量兼容中文文件名，但如果你的终端编码显示异常，建议直接在资源管理器中复制绝对路径后再运行。
